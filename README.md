# LSUV Service Website

Static website for GitHub hosting with minimal maintenance.

## Included Features

- Sticky navigation with links to Home, Gallery, Team, and Contact
- Hero section with direct call-to-action button
- Team profile grid
- Gallery with lightbox (click image to expand)
- Service request form
- Discord webhook submission from the contact form

## Webhook Setup

Open script.js and update DISCORD_WEBHOOK_URL if needed.

The form sends these fields to Discord:

- Name
- Email or Phone
- Subject
- Message

## Hosting

Deploy directly with GitHub Pages.
No backend, database, or regular maintenance required.
